# MNIST Linear Regression Project

This project trains a simple Linear Regression model from scratch on MNIST CSV data.

## Dataset Files Needed

Put these two files in this folder:

- `mnist_train_small.csv`
- `mnist_test.csv`

Each CSV should have the label in the first column and 784 pixel columns after it.

## Run

```powershell
python mnist_linear_regression_from_scratch.py
```

The script saves:

- `cost_over_iterations.png`
- `sample_digit_predictions.png`
